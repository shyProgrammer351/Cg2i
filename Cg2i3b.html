문제

10진수를 입력받아 2진수로 변환하여 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
십진수를 입력하세요: 6
(십진수)6 = (이진수)110 


십진수를 입력하세요: 35
(십진수)35 = (이진수)100011 


십진수를 입력하세요: 300
(십진수)300 = (이진수)100101100

코드 빈칸 채우기

#include <stdio.h> 
 
void ( int num ) { 
	 ( num  2 )  
		( num  2 ); 
	printf( "%d", num  2 ); 
} 
 
 
int main() { 
	printf( "십진수를 입력하세요: " ); 
	int num = 0; 
	scanf( "%d", &num ); 
 
	printf( "(십진수)%d = (이진수)", num ); 
	decimal2binary( num ); 
 
	return 0; 
}
