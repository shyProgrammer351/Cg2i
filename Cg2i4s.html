문제

파일에서 야구 경기 정보를 읽어들여 그 결과를 바탕으로 스포츠 뉴스를 출력하는 문제입니다 실행순서를 클릭하세요 참고로, 야구 경기 정보는 아래와 같이 '정보 [ 탭 ] 정보 [ 탭 ] 정보 [ 탭 ] ...'의 구조로 저장되어 있습니다. 

--------- 파일 baseball1.txt의 내용 --------- 
15	롯데	KT	10	1	김승회	정대현	정훈	6회초1아웃	2	박진형
18	한화	SK	8	4	정우람	문승원	강경학	6회초1아웃	3	김재영
21	LG	KIA	6	1	오석주	강이준	양원혁	2회초2아웃	2	오지환
--------- 파일 baseball2.txt의 내용 --------- 
18	한화	SK	8	4	정우람	문승원	강경학	6회초1아웃	3	김재영
--------- 파일 baseball3.txt의 내용 --------- 
21	LG	KIA	6	1	오석주	강이준	양원혁	2회초2아웃	2	오지환


관련신문기사 - http://news1.kr/articles/?2246990
파일이름을 입력해 주세요 : baseball1.txt
( KT 1 : 롯데 10 )
KT는 15일 열린 프로야구 롯데와의 홈 경기에서 1-10점으로 크게 패하며 홈 팬들을 실망시켰다. 
KT는 정대현을 선발로 등판시켰고 롯데는 김승회가 나섰다. 롯데는 정훈이 맹활약을 펼쳤다. 
정훈은 6회초1아웃에 맞이한 타석에서 2점을 뽑아내며, 롯데의 9점차 승리를 이끈 일등공신이 됐다. 
KT는 박진형을 끝까지 공략하지 못하며 안방에서 롯데에 9점차 승리를 내주었다.
( SK 4 : 한화 8 )
SK는 18일 열린 프로야구 한화와의 홈 경기에서 4-8점으로 크게 패하며 홈 팬들을 실망시켰다. 
SK는 문승원을 선발로 등판시켰고 한화는 정우람가 나섰다. 한화는 강경학이 맹활약을 펼쳤다. 
강경학은 6회초1아웃에 맞이한 타석에서 3점을 뽑아내며, 한화의 4점차 승리를 이끈 일등공신이 됐다. 
SK는 김재영을 끝까지 공략하지 못하며 안방에서 한화에 4점차 승리를 내주었다.
( KIA 1 : LG 6 )
KIA는 21일 열린 프로야구 LG와의 홈 경기에서 1-6점으로 크게 패하며 홈 팬들을 실망시켰다. 
KIA는 강이준을 선발로 등판시켰고 LG는 오석주가 나섰다. LG는 양원혁이 맹활약을 펼쳤다. 
양원혁은 2회초2아웃에 맞이한 타석에서 2점을 뽑아내며, LG의 5점차 승리를 이끈 일등공신이 됐다. 
KIA는 오지환을 끝까지 공략하지 못하며 안방에서 LG에 5점차 승리를 내주었다.

프로그램 코드

	#include <stdio.h>


1	int main() {

		char date[1024]="", winningTeam[1024]="", losingTeam[1024]="";

		int  winningPoint=0, losingPoint=0, bestPointScore=0;

		char winningPitcher[1024]="", losingPitcher[1024]="";

		char bestPlayer[1024]="", bestPlayPoint[1024]="", missedPlayer[1024]="";

		char filename[1024]="", line[1024]="";

2		FILE *inFile = NULL;


		printf( "파일이름을 입력해 주세요 : " );

		scanf( "%s", filename );


3		if ( ( inFile = fopen( filename, "rt" ) ) == NULL ) {

			printf( "[오류] 파일을 열 수 없습니다!" );

4			return -1;

		}


5		while ( fgets( line, 1024, inFile ) != NULL ) {

6			if( sscanf( line, "%s\t%s\t%s\t%d\t%d\t%s\t%s\t%s\t%s\t%d\t%s\n",

				date, winningTeam, losingTeam, &winningPoint, &losingPoint,

				winningPitcher, losingPitcher,

				bestPlayer, bestPlayPoint, &bestPointScore, missedPlayer ) == 0 ) {

				printf( "[오류] 한 경기씩 주요 경기 내용 읽어올 수 없습니다." );

7				break;

			}


			printf( "\n\n( %s %d : %s %d )\n",

				losingTeam, losingPoint, winningTeam, winningPoint );

			printf( "%s는 %s일 열린 프로야구 %s와의 홈 경기에서 %d-%d점으로 크게 패하며 ",

				losingTeam, date, winningTeam, losingPoint, winningPoint );

			printf( "홈 팬들을 실망시켰다. \n%s는 %s을 선발로 등판시켰고 %s는 %s가 나섰다. ",

			 losingTeam, losingPitcher, winningTeam, winningPitcher );

			printf( "%s는 %s이 맹활약을 펼쳤다. \n%s은 %s에 맞이한 타석에서 %d점을 뽑아내며, ",

				winningTeam, bestPlayer, bestPlayer, bestPlayPoint, bestPointScore );

			printf( "%s의 %d점차 승리를 이끈 일등공신이 됐다. \n%s는 %s을 끝까지 공략하지 못하며 ",

				winningTeam, ( winningPoint - losingPoint ), losingTeam, missedPlayer );

8			printf( "안방에서 %s에 %d점차 승리를 내주었다.", winningTeam, ( winningPoint - losingPoint ) );

		}


9		fclose( inFile );


10		return 0;

	}
